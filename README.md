# Java Bank Account Assignment

Репозиторий содержит решение практического задания №1 по Java (ООП. Классы. Операторы).

## Задание
Необходимо было создать класс `BankAccount` со следующими характеристиками:

**Поля:**
- Имя владельца
- Баланс
- Дата открытия
- Статус блокировки
- Номер счета (генерируется случайно)

**Методы:**
- `deposit(amount)` - пополнение счета
- `withdraw(amount)` - снятие денег
- `transfer(otherAccount, amount)` - перевод на другой счет
- `toString()`, `equals()`, `hashCode()`

## Выполнение
- Основная логика реализована в [`src/BankAccount.java`](src/BankAccount.java)
- Тестирование всех методов и проверок (недостаточно средств, заблокированный счет) реализовано в [`src/Main.java`](src/Main.java)
